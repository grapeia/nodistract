# 📖 NoDistract

**NoDistract** é um modo leitura limpo e minimalista para qualquer página web. Remove anúncios, menus, barras laterais e outros elementos de distração, apresentando apenas o conteúdo principal do artigo com uma tipografia elegante e agradável aos olhos.

## ✨ Características

- 🎯 **Extração inteligente de conteúdo** - Usa Mozilla Readability para identificar o conteúdo principal
- 🎨 **Design minimalista** - Interface limpa com gradiente suave e tipografia otimizada
- 📱 **Responsivo** - Funciona perfeitamente em desktop, tablet e mobile
- 🖼️ **Preserva imagens** - Mantém imagens do artigo com tratamento adequado
- 🎬 **Preserva vídeos** - Detecta players (`<video>` e iframes de embed) e os mantém no modo leitura, bloqueando pop-ups do player via sandbox
- ⚡ **Sem dependências pesadas** - Apenas Readability.js, sem jQuery ou Bootstrap
- 🌙 **Modo escuro** - Alterna entre tema claro e escuro, respeitando a preferência do sistema
- 🔠 **Tamanho de fonte ajustável** - Botões A− / A+ com preferência salva entre sessões
- 🔄 **Reversível** - Botão para recarregar a página original a qualquer momento
- 📖 **Pronto para impressão** - Estilos otimizados para impressão

## 🚀 Demo

Acesse [https://grapeia.github.io/nodistract/demo.html](https://grapeia.github.io/nodistract/demo.html) para ver o NoDistract em ação!

A página de demonstração contém:
- Artigo completo sobre leitura focada
- Diversos elementos de distração (anúncios, menus, sidebars)
- Botão para ativar o modo leitura e ver a transformação instantânea

**Como usar a demo:**
1. Abra `demo.html` no navegador
2. Clique no botão "🚀 Ativar Modo Leitura"
3. Veja a página transformada em uma experiência de leitura limpa

## 💾 Instalação

Acesse [https://grapeia.github.io/nodistract/install.html](https://grapeia.github.io/nodistract/install.html) para instalar o bookmarklet!

A página de instalação oferece:
- **Método 1:** Arraste o botão para sua barra de favoritos
- **Método 2:** Copie e cole o código manualmente nos favoritos
- **Teste instantâneo** na própria página de instalação
- Estatísticas sobre o tamanho do código

**Como instalar:**
1. Abra `install.html` no navegador
2. Arraste o botão "📖 NoDistract" para sua barra de favoritos
3. Navegue até qualquer artigo ou página com texto
4. Clique no favorito "NoDistract" para ativar o modo leitura
